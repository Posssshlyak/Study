Обучение HTML: Крайне полезная информация для вас собрана здесь

Чтобы создавать веб-сайты, вы должны знать о HTML — фундаментальной технологии, которая используется для определения структуры веб-страницы. HTML применяется для того, чтобы определить как должен отображаться ваш контент: в виде абзаца, списка, заголовка, ссылки, изображения, мультимедийного проигрывателя, формы или же в виде одного из множества других доступных элементов, а также возможного нового элемента, который вы сами создадите.

"JavaScript", и как его использовать для придания динамической функциональности веб-страницам (например, определение вашего местоположения и отображение его на карте, создание элементов, которые будут появляться/исчезать при нажатии на кнопку, сохранение данных пользователей локально на их компьютерах и многое другое). Решение общих HTML задач

Использование HTML для решения общих задач содержит ссылки на разделы контента, объясняющего, как использовать HTML для решения очень распространённых проблем при создании веб-страницы: работа с заголовками, добавление изображений или видео, выделение содержимого, создание базовой формы и т.д.

Примеры работы

Скелет тела HTML:

<!DOCTYPE html>

<html lang="ru">

<head>

</head>

<body>

</body>

</html>

<!DOCTYPE> - это не тег, а обязательная инструкция объявления типа документа. Она нужна для того, чтобы сообщить браузеру в какой версии HTML написан документ.

Далее видно, что указывается язык страницы, его можно указывать практически любой, а дальше браузер будет распознавать, возможна ли поддержка данного языка

<HEAD> - это раздел в который будет записываться основные элементы, теги, которые будут связывать файлы и определять некоторые моменты.

<BODY> - необходим для ввода тегов и информации, которая будет выводиться на странице.

Собственно, это и есть основные теги для тела кода HTML, об остальных же особенностях можно узнать, нажав на Узнать

Ссылка направляет вас на сторонний ресурс, где вы можете найти и другую значимую для вас информацию

Пример создания таблицы:

<table>

<thead>

<tr>

<th> 1ая ячейка </th>

<th> 2ая ячейка </th>

</tr>

</thead>

<tbody>

<tr>

<td> 1ая ячейка </td>

<td> 2ая ячейка </td>

</tr>

<tr>

<td> 1ая ячейка</td>

<td>2ая ячейка </td>

</tr>

</tbody>

</table>

1ая ячейка 2ая ячейка
1ая ячейка 2ая ячейка
1ая ячейка 2ая ячейка

Здесь представлена таблица, состоящая из двух столбцов и трех строк.Оглавление происходит с помощью <table>

Элемент <thead> предназначен для хранения одной или нескольких строк, которые представлены вверху таблицы.

Элемент <tbody> предназначен для хранения одной или нескольких строк таблицы. Это позволяет создавать структурные блоки, к которым можно применять единое оформление через стили, а также управлять их видом

Тег <tr> служит контейнером для создания строки таблицы.

Предназначен для создания одной ячейки таблицы. Тег <td> должен размещаться внутри контейнера <tr>, который в свою очередь располагается внутри тега <table>.
